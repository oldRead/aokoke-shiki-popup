4枚の写真を選択すると、トップページにポップアップ表示します。

== Description ==

Aokoke Shiki-Popupは4枚の写真を選択し、トップページにポップアップ表示するプラグインです。
選択した写真はリサイズされて配置されるので、異なるサイズの画像を指定することができます。

***設定項目は管理メニューの「設定」に内包されます。***

## 諸注意！
>現在、画像の枚数に合わせて画像サイズを自動調節する機能がありません。必ず4枚の画像を指定する必要があります。
>また、ポップアップをトップページ以外に表示することはできませんので、ご了承ください。

## 機能紹介

### 1. メッセージ
ポップアップの中央に表示するメッセージを指定することができます。

### 2. 再表示制限
再表示に制限をかけることができます。

### 3. リンク
ポップアップを表示した際にジャンプするリンクを指定することができます。

### 4. 写真の設定
画像URLを入力するか、画像ライブラリから選択することができます。
各画像には短いタイトルを入力し、重ねて表示することができます。

== Installation ==

1. Upload `AokokeShikiPopup` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
